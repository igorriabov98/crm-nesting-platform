# Паспорт эталона СТВ-300 «Стол-ванна» (двойник изделия класса ЛЕДА.024)

Синтетический двойник реального изделия: та же конструктивная схема,
«круглые» размеры, все значения аналитические (объёмы сверены 0.0000%).
7 одиночных STEP + сборка (17 тел) + 6 PDF в стиле оригинала.

## Новые паттерны, которых не было в прошлых эталонах
1. **Развёртка в ПРИМЕЧАНИЯХ, не в штампе** («3. Развертка − AхB мм.») —
   тест извлечения из notes.
2. **Толщина в сортаменте**: «Лист БТ-ПН-2,0 ГОСТ 19903-90» → t=2.0;
   «БТ-ПН-8» → t=8. Марка отдельной строкой (Ст3пс).
3. **Исполнения**: один чертёж СТВ-300.04 с таблицей (базовое/-01/-02),
   три разных STEP. Матчер должен развести три длины по одной таблице.
4. **Ванна с замкнутыми углами** (сварная, цикл стенок): развернуть
   нельзя в принципе — истина о развёртке ТОЛЬКО в PDF.
5. **Толстый гиб t8 r8** (ножка) — прежний максимум был t3.

## КЛЮЧЕВОЙ КЕЙС: ванна СТВ-300.01
PDF-примечание задаёт развёртку **1340 x 890** (сварная технология:
1100+2·120 x 650+2·120). STEP-геометрия ванны НЕ разворачивается
(стенки замкнуты в цикл). Возможные исходы парсинга:
- contour-extractor возьмёт пару граней дна → EXACT_BREP 1100 x 650;
- либо fallback.
В ОБОИХ случаях правильное поведение системы:
1. auto-apply размеров из PDF ЗАБЛОКИРОВАН dimension-guard'ом
   (1340x890 против 1100x650 — расхождение >> 2%);
2. dimensionMismatch = true с пояснением;
3. применение только вручную с force (осознанное решение технолога).
Молчаливое применение 1340x890 БЕЗ флага = провал теста.
Это точная модель реальной обшивки ЛЕДА.024.00.001 (развёртка
1172x1186 в примечании при габарите 790x360x1186).

## Сборка STV-300-SB (17 тел)
1 ванна + 2 стенки + 4 ножки + 2+2+2 уголка (три исполнения!) +
4 стойки. Спецификация сборочного PDF содержит РАЗНЫЕ позиции для
исполнений (5/6/7) — матчер обязан развести уголки по длинам
(1150/700/1080), а не слить в одну позицию по одинаковому профилю.
Масса сборки 37.26 кг.

## PDF
Все векторные с текстовым слоем. Материал двумя строками
(сортамент + марка), развёртки гнутых — в примечаниях, у ванны и
ножки в штампе развёртки НЕТ. Групповой чертёж уголка — таблица
исполнений с массами и развёртками (54,6 в таблице = 54.60 точно).
